# Market Intelligence Frameworks

> *The methodology behind vendor analysis and category sourcing should not be a black box.*

Welcome to my open-source repository of market intelligence frameworks.

As a Market Intelligence Consultant, I believe procurement teams perform better when they have access to clear, repeatable methods for decoding categories. This repository contains high-level, anonymized templates and markdown frameworks that I use in my daily practice — distilled into a form anyone can adapt.

## Frameworks (Coming Soon)

The following frameworks will be added progressively. Each is grounded in primary data and refined through real-world category briefs.

| Framework | Purpose |
| :--- | :--- |
| **The Supplier Landscape Matrix** | A structured way to map vendors by capability, market presence, and risk. |
| **The Spend Narrative Template** | How to structure a category brief so that it reads like a story — and drives action. |
| **The Authentic Source Checklist** | A QC checklist for verifying primary data sources before they enter a briefing. |
| **The Negotiation Brief Outline** | A pre-meeting structure for sourcing managers preparing for high-stakes negotiations. |
| **The Category Risk Heatmap** | A repeatable format for visualizing supplier and macro risk in a single page. |

## Philosophy

> *Primary data first. Frameworks second. Narrative last.*

These frameworks are designed to be cloned, adapted, and applied to your specific procurement categories. They contain no proprietary or company-specific data. If you find them useful — or if you'd like to contribute your own — please open an issue or reach out via my [website](https://tathastunaranje.github.io).
